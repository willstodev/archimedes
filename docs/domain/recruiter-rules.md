# Recruiter Rules

Archimedes identifies relevant public recruiter profiles for human review. It must not automate LinkedIn login, private scraping, connection requests, or messages.

Prioritize recruiters who are:

- Connected to companies with selected or high-scoring jobs.
- In technical recruiting, engineering recruiting, talent acquisition, or sourcing roles.
- Associated with software engineering hiring.
- Connected to the configured target geography, remote, international, or global hiring.

Outreach drafts may be prepared only after deterministic relevance checks pass. Outreach status can become `sent` only after explicit user confirmation.
