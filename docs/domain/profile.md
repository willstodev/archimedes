# Target Profile

Canonical profile rules live in [AGENTS.md](../../AGENTS.md). This file exists to keep implementation work focused without copying the full constitution into task context.

The default target opportunities are senior backend or full-stack software engineering roles aligned with:

- TypeScript, Node.js, NestJS, Angular, and AWS.
- Distributed systems, event-driven architecture, APIs, and cloud-native systems.
- Remote-friendly international eligibility.
- Compensation above the locally configured floor.

This document should only contain durable profile facts needed by deterministic filters and scoring. Private resume contents, generated reports, and mutable search state do not belong here.
